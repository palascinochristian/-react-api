Ah, capisco! Il problema è che quando i tags vengono stampati dal backend (o dal frontend), sono un array, ma probabilmente vengono visualizzati come una stringa perché React cerca di renderizzare il valore come tale.

Per visualizzare correttamente i tags come una lista di valori separati da virgola (o come preferisci), puoi fare un mapping del tuo array di tags per stampare ogni elemento separato da una virgola, o usarli in modo più chiaro. Ecco come risolvere il problema:

jsx

  Tags: {recipe.tags.join(", ")}
</div>

Spiegazione:

  • recipe.tags.join(", "): Questo unirà gli elementi dell'array tags in una stringa, separandoli con una virgola e uno spazio.
  • Se preferisci visualizzare i tags in un altro modo (ad esempio, come una lista separata o su più righe), puoi fare qualcosa del tipo:
jsx
<div className="recipe-tags">

  <ul>
